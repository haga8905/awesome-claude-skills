# Awesome Claude Skills

A collection of skills for [Claude Code](https://docs.anthropic.com/en/docs/claude-code) — Anthropic's agentic coding tool.

## Skills

### Stock Analysis (`stock-analysis.skill`)

A multi-agent equity research framework (v2.0) that orchestrates 5 specialized agents to produce institutional-grade stock analysis across 10 phases. Supports full analysis, quick scans, head-to-head comparisons, and watchlist updates.

### Development Team (`development-team.skill`)

A structured 4-agent development framework: **Architect → Developer → Reviewer → Tester**. Each agent has defined roles, outputs, and handoff conditions with built-in quality gates and feedback loops.

## Installation

1. Download the `.skill` file(s) you want
2. Place them in your `~/.claude/skills/` directory (or project-level `.claude/skills/`)
3. The skills will be available in Claude Code automatically
